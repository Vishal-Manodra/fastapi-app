# FastAPI Todo API

A simple REST API built with FastAPI and Python.

## Features

- Create items
- List items
- Get item by ID
- Request validation with Pydantic
- Error handling with HTTPException

## Tech Stack

- Python 3.13
- FastAPI
- Uvicorn
- Pydantic
